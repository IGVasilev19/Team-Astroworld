# Тема
Нашата идея за проекта е да направим приложение (“Drivey”), което използва дронове, които да събират информация патрулирайки по улиците. Тази информация ще включва свободни места за паркиране и неправилно паркирали автомобили. В случай, че някой е паркирал неправилно, дронът ще заснеме номера на колата и ще изпрати координати на влекачите, за да може да се вземат необходимите мерки за справянето с проблема. Тези шофьори, които са се регистрирали в приложението, освен че ще получават сигнали за свободни места за паркиране, ще получат и известие, в случай, че са нарушили някакви правила и ще подлежат на глоба.
